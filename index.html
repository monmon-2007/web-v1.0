Sudoku Solver

Sudoku solver solves sudoku using backtracking algorithem. The backtracking algorithm enumerates a set of partial candidates that, in principle, could be completed in various ways to give all the possible solutions to the given problem. The completion is done incrementally, by a sequence of candidate extension steps.

Reliable UDP File Transfer Protocol

UDP File Transfer Protocol is a reliable communication protocol on top of an inherently unreliable communication fabric from one machine to many machines. It implements fast, reliable one-way communication from a single sender (the first program) running on one host to multiple receivers (the second program) running on different hosts in an unreliable environment.
